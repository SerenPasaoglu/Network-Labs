DHCP Lab

The purpose of this lab is to understanda how DHCP automatically assigns IP addresses to computers
**DHCP**: It is a management protocol that automatically and centrally distributes necessary network configuration, such as IP address, 
and DNS information, to devices connected to the network.

## Topology

-9 Laptops
-3 printers
-1 switch
-1 DHCP server

## Configuration

- Configured the DHCP server
- Created a DHCP IP address pool
- Connected computer and printers to the network
- Enabled automatic IP assignment

  ## Testing

  The clients successfully obtained IP addresses automatically from the DHCP serves

  ## Result

  I've learned how DHCP simplified network administration via automatically assigning IP addresses.
  I learned the basics of DHCP configuration, address pools, and client-server communication.
